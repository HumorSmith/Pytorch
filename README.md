# 环境安装

地址:https://pytorch.org/get-started/locally/

conda命令行:
- 安装pytorch
```angular2html
conda install pytorch::pytorch torchvision torchaudio -c pytorch
```
- 安装tensorboard(查看训练数据)
```shell
conda install tensorboard
```
- tensorboard使用
```shell
 tensorboard --logdir=logs
```
- 安装opencv
```shell
conda install opencv
```
